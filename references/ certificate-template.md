# QC Certificate Template

Generate an HTML file with this structure, then convert to PDF.

## HTML Structure

```html
<!DOCTYPE html>
<html>
<head>
<style>
  body { font-family: system-ui, sans-serif; color: #1a1a2e; max-width: 800px; margin: 0 auto; padding: 40px; }
  .header { border-bottom: 3px solid #302b63; padding-bottom: 16px; margin-bottom: 24px; }
  .header h1 { font-size: 22px; color: #302b63; margin: 0; }
  .meta { font-size: 11px; color: #666; margin-top: 8px; }
  .summary-bar { display: flex; gap: 16px; margin: 20px 0; }
  .stat { padding: 12px 20px; border-radius: 8px; font-weight: 600; font-size: 14px; }
  .stat-pass { background: #f0fdf4; color: #16a34a; border: 1px solid #bbf7d0; }
  .stat-fail { background: #fef2f2; color: #dc2626; border: 1px solid #fecaca; }
  .stat-review { background: #fffbeb; color: #d97706; border: 1px solid #fde68a; }
  table { width: 100%; border-collapse: collapse; margin: 16px 0; font-size: 11px; }
  th { background: #302b63; color: #fff; padding: 8px 12px; text-align: left; }
  td { padding: 8px 12px; border-bottom: 1px solid #e5e5ee; vertical-align: top; }
  .pass { color: #16a34a; font-weight: 600; }
  .fail { color: #dc2626; font-weight: 600; }
  .review { color: #d97706; font-weight: 600; }
  .evidence { font-size: 10px; color: #555; margin-top: 4px; }
  .footer { margin-top: 32px; padding-top: 16px; border-top: 2px solid #e5e5ee; font-size: 10px; color: #888; }
  .agreement { font-size: 10px; padding: 2px 6px; border-radius: 3px; }
  .agree { background: #f0fdf4; color: #16a34a; }
  .disagree { background: #fffbeb; color: #d97706; }
</style>
</head>
<body>

<div class="header">
  <h1>🔍 QC Certificate</h1>
  <div class="meta">
    Deliverable: [NAME] | Type: [TYPE] | Date: [DATE]<br>
    Mode: [single/cross] | Agent A: [MODEL] | Agent B: [MODEL]<br>
    Depth: [quick/standard/deep] | Checklist: [standard/custom]
  </div>
</div>

<div class="summary-bar">
  <div class="stat stat-pass">✅ [X] Passed</div>
  <div class="stat stat-fail">❌ [Y] Failed</div>
  <div class="stat stat-review">⚠️ [Z] Review</div>
</div>

<!-- RESULTS TABLE -->
<table>
  <tr>
    <th>Item</th>
    <th>Agent A</th>
    <th>Agent B</th>
    <th>Result</th>
  </tr>
  <!-- For each checklist item: -->
  <tr>
    <td>[Checklist item name]</td>
    <td><span class="pass">PASS</span><div class="evidence">[Agent A evidence]</div></td>
    <td><span class="pass">PASS</span><div class="evidence">[Agent B evidence]</div></td>
    <td><span class="agreement agree">AGREE</span></td>
  </tr>
  <!-- Repeat for all items -->
</table>

<!-- HUMAN REVIEW SECTION (only if YELLOW items exist) -->
<h2>⚠️ Items for Human Review</h2>
<p>The following items had disagreements or could not be verified. Manual review recommended.</p>
<table>
  <tr><th>Item</th><th>Agent A says</th><th>Agent B says</th><th>Suggested check</th></tr>
  <!-- List YELLOW items with both agents' evidence and a specific human action -->
</table>

<div class="footer">
  Generated by Adversarial QC v1.0 | [TIMESTAMP UTC]<br>
  This certificate documents the automated review process. It does not guarantee correctness — human review of flagged items is recommended.
</div>

</body>
</html>
```

## PDF Conversion

Convert using Chromium headless:
```bash
chromium --headless --disable-gpu --no-sandbox --print-to-pdf=output.pdf --print-to-pdf-no-header input.html
```

Or use a local HTTP server if the HTML has external resources.
